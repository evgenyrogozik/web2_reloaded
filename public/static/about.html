CITS3403 Project 2

Author: Evgeny Rogozik / Wilsen Lie

Student number: 21177069 / 21105559

This implemented version of Connect Four game can be played both versus a friend or a computer. However it is fair to mention that in the current version of the game computer just makes random moves without considering any possible win movements. Research has been done on which AI algorithm is used for this particular game. Minimax Algorithm can be implemented for this game's AI but that would rather be a project for CITS3001 unit.
